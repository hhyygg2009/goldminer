### 技术选型

- Cocos Creator版本：2.4.5
- 可以编译到微信小游戏


### 项目布局

``` lua
src -- 源码目录
└── assets -- 资源文件
```

## 搭建步骤
- 安装Cocos Creator，安装对应版本编辑器
- 导入项目
